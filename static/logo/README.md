# PWA 로고 이미지

이 폴더의 이미지는 **PWA(홈 화면 추가, 스플래시, 푸시 알림)** 에 사용됩니다.

## 필수 파일

| 파일 | 용도 | 권장 크기 |
|------|------|-----------|
| **side1.jpg** | 기본 로고 (manifest, Apple 홈 화면, 푸시 알림 아이콘) | 정사각형, 512px 이상 권장 |

`side1.jpg` 하나만 있어도 동작합니다.  
manifest에서는 192×192, 512×512 로 선언되며, Apple은 180×180 등으로 참조합니다.

## 선택 파일 (권장)

PNG를 넣으면 manifest에서 자동으로 사용됩니다.

| 파일 | 용도 |
|------|------|
| **icon-192.png** | 192×192 픽셀, 투명 또는 단색 배경 |
| **icon-512.png** | 512×512 픽셀 (스플래시·설치 화면) |

- **maskable**: Android에서 원형/둥근 아이콘으로 잘리므로, 중요한 내용은 중앙 80% 안에 두는 것이 좋습니다.
- PNG가 없으면 `side1.jpg`가 192/512 용도로 사용됩니다.

## 참조 위치

- **manifest.json**: `/manifest.json` (아이콘 목록)
- **Apple 홈 화면**: `<link rel="apple-touch-icon">` (120, 152, 180)
- **푸시 알림**: `static/sw.js` (icon, badge → `side1.jpg`)
