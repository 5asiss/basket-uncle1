# 이미지 대량 업로드 후 확인 방법

관리자 **대량등록** 탭에서 「이미지 올리기」로 여러 장을 올린 뒤, 아래 순서로 확인하면 됩니다.

---

## 1. 업로드 직후

- **플래시 메시지** 확인  
  - 예: `N개 이미지가 static/uploads/에 저장되었습니다. 엑셀의 대표·상세이미지파일명에 위 파일명을 그대로 입력하세요.`
- 같은 파일명이 이미 있으면: `같은 이름의 파일이 이미 있어 M개는 건너뛰었습니다.` 로 일부만 저장될 수 있음.

---

## 2. 저장된 이미지 목록에서 확인

- **대량등록** 탭 하단 **「이미지 대량 업로드 후 확인 방법」** 섹션을 봅니다.
- **저장된 이미지 목록 (최근 100개, 최신순)** 에 방금 올린 **파일명**이 있는지 확인합니다.
- 파일명을 클릭하면 `/static/uploads/파일명` 으로 열려 **미리보기**로 확인할 수 있습니다.

---

## 3. 엑셀에 반영 후 상품 업로드

- 엑셀 양식의 **대표이미지파일명**·**상세이미지파일명** 란에 위 목록에 보이는 **파일명만** 그대로 입력합니다.
- 대표 = 접미사 `_1`, 상세 = `_2` ~ `_10` (쉼표 구분 가능).
- **상품 업로드** 버튼으로 엑셀을 제출합니다.

---

## 4. 상품에 이미지가 붙었는지 확인

- **상품관리** 탭으로 이동합니다.
- 해당 상품 행의 **사진** 칸을 보거나, **수정**으로 들어가서 대표·상세 이미지가 노출되는지 확인합니다.

---

## 요약

| 단계 | 확인 내용 |
|------|-----------|
| 1 | 플래시 메시지에서 «N개 저장» 확인 |
| 2 | 대량등록 탭 하단 **저장된 이미지 목록**에서 파일명·미리보기 확인 |
| 3 | 엑셀에 파일명 입력 후 상품 업로드 |
| 4 | 상품관리에서 해당 상품 이미지 노출 확인 |

저장 위치: **`static/uploads/`** (서버 기준).  
파일명은 **대소문자·공백·특수문자**까지 엑셀에 **완전히 동일하게** 입력해야 합니다.
